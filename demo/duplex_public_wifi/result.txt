사용자가 "gogogo"라고 했으니 워크스페이스 설정으로 진행하겠다는 의미로 보인다. 단계별로 명확하게 안내해주자.좋아요! 바로 시작합시다!
1. User Settings (전역 설정)
Ctrl+Shift+P → Preferences: Open User Settings (JSON)
내용을 다음으로 변경:
json{
    "code-runner.runInTerminal": true,
    "code-runner.clearPreviousOutput": true,
    "code-runner.saveFileBeforeRun": true
}
Ctrl+S 저장
2. Workspace Settings (프로젝트 설정)
Ctrl+Shift+P → Preferences: Open Workspace Settings (JSON)
다음 내용 입력:
json{
    "python.defaultInterpreterPath": "${workspaceFolder}/venv_denoiser/bin/python",
    "code-runner.executorMap": {
        "python": "${workspaceFolder}/venv_denoiser/bin/python -u"
    }
}
Ctrl+S 저장
3. 테스트
bashcd ~/Facebook-Denoiser-in-Raspberry-Pi-5
echo 'import sys; print(sys.executable)' > test_venv.py
VSCode에서:

test_venv.py 열기
▶️ Run Code 클릭
출력에 /venv_denoiser/bin/python 경로가 나오는지 확인
